# DSH Hacks V1 — Devpost Submission

## Project Name
**AXIOM** — Adaptive eXplanatory Intelligence via Orthogonal Modeling

## Tagline
Compute First. Explain Second.

## Elevator Pitch (≤500 chars)
AXIOM fixes broken AI STEM tutors by separating truth from explanation. Layer 1 is a deterministic combinatorial decomposer (Punnett grids, permutations, electron configs, harmonics) that computes exact answers in TypeScript. Layer 2 is a narrative adapter (SuperGrok / xAI Grok) that only explains the verified JSON — never guesses math. Integrated with RadhikaChain gamification (XP, Bhakti, badges).

## Problem Statement
AI tutors sound confident but get STEM math wrong. Students learn incorrect probabilities, formulas, and configurations. Engagement without accuracy undermines trust.

## Solution
Dual-engine architecture:
1. **Combinatorial Decomposer** — pure math, instant, offline-capable
2. **Narrative Adapter** — AI explains structure with hooks, analogies, Socratic questions

## Theme Alignment
**AI × STEM Education** — uses AI only where it excels (communication), not where it fails (exact computation).
