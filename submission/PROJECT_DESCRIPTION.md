# AXIOM AI — Dual-Engine Combinatoria
## One-Page Project Description (DSH Hacks V1)

### Purpose
AXIOM is an AI-powered STEM tutor for students aged 13–18 that solves the #1 flaw of existing AI tutors: **mathematical inaccuracy**. Instead of asking an LLM to both calculate and explain, AXIOM separates these into two orthogonal layers — **compute first, explain second**.

### Key Features

**Layer 1: Combinatorial Decomposer (Deterministic)**  
Pure TypeScript, zero network, instant exact JSON.
- **Genetics:** Full gamete enumeration → Punnett grid → exact genotype probabilities
- **Math:** Permutations & combinations with formula, steps, and small-n examples
- **Chemistry:** Aufbau principle → electron configuration + noble-gas notation + orbital diagram
- **Physics:** Harmonic series → frequency, wavelength, musical note per harmonic

**Layer 2: Narrative Adapter (AI)**  
SuperGrok (xAI) transforms the verified JSON into vivid, Socratic explanations for ages 13–18.  
Fallbacks: RadhikaChain agent-core and high-quality demo narratives (works offline / without API keys).

**Gamification & Ecosystem**  
XP, levels, domain mastery badges, learning streaks + optional Bhakti confidence tier from RadhikaChain wallet.

### Innovation
Inspired by Piṅgala’s *prastāra* and the Meru-prastāra (ancient combinatorial triangle). STEM problems are treated as structures to be enumerated exactly before any narrative is generated. This is how expert human tutors actually work.

### Impact
Students see **structure before story**, building deep, trustworthy understanding. Teachers can trust the math. The same combinatorial foundation powers future extensions into arbitrage path enumeration (Samsara Core) and Jyotish combinatorial analysis.
