# AXIOM — One Page Project Description

## Purpose
AXIOM is an AI-powered STEM tutor for students aged 13–18 that solves the #1 flaw of existing AI tutors: **mathematical inaccuracy**. Instead of asking an LLM to both calculate and explain, AXIOM separates these into two orthogonal layers.

## Key Features

### Layer 1: Combinatorial Decomposer (Deterministic)
- **Genetics:** Full gamete enumeration → Punnett grid → exact genotype probabilities
- **Math:** Permutations & combinations with formula, steps, and small-n examples
- **Chemistry:** Aufbau principle → electron configuration + noble-gas notation + orbital diagram
- **Physics:** Harmonic series → frequency, wavelength, musical note per harmonic

### Layer 2: Narrative Adapter (AI)
- Anthropic Claude 3.5 Sonnet transforms exact JSON into vivid explanations
- Fallback: RadhikaChain agent-core (Bedrock/Workers AI)
- Demo mode: high-quality pre-written narratives (no API key needed)

### Gamification (RadhikaChain Ecosystem)
- XP, levels, domain mastery badges, learning streaks
- Wallet connect → Bhakti confidence tier from RadhikaChain D1
- White Hat motivation design (mastery, not punitive leaderboards)

## Innovation
Inspired by Jyotish Sragdharā meter decomposition — treating STEM problems as combinatorial structures to enumerate exactly before narrating. This is how expert human tutors work: compute first, explain second.

## Impact
Students see **structure before story**, building deep understanding. Teachers can trust the math. The architecture scales to population genetics, quantum states, signal processing, and more domains.
