# AXIOM — 90-second demo script

## 0–10 seconds — the promise

**On screen:** AXIOM landing page.

**Say:** “AXIOM computes STEM answers with deterministic code, then uses OpenAI to explain only the verified result.”

## 10–30 seconds — compute

**On screen:** Genetics selected. Inputs Aa and aa. Click “Compute deterministically.”

**Say:** “This is the primary demo: a single-gene Aa × aa cross. Before any AI text appears, AXIOM enumerates the exact 2×2 Punnett grid.”

**Show:** The four cells and the Aa / aa percentage chips.

## 30–45 seconds — inspect evidence

**On screen:** Open the Evidence ledger.

**Say:** “The result is packaged as stable facts and steps. Notice the IDs such as genetics.probability.Aa. Those are the only evidence IDs the explanation may cite.”

## 45–65 seconds — explain

**On screen:** Choose High school and English. Click “Generate explanation.”

**Say:** “The server recomputes the deterministic result, sends that record to OpenAI, validates the JSON, and then verifies the claims. OpenAI adapts the wording; it does not calculate the cross.”

**Show:** Citation chips linking into the ledger.

## 65–75 seconds — verify honestly

**On screen:** Verification badge and narrative footer.

**Say:** “A green badge means the cited output passed AXIOM’s checks. If a citation is missing, a number changes, or model output is malformed, AXIOM shows Warnings or Could not verify.”

## 75–85 seconds — fallback

**On screen:** Run without OpenAI variables, or point out the deterministic fallback indicator.

**Say:** “No API key? The learning flow still works. AXIOM shows a cited deterministic fallback and labels it clearly.”

## 85–90 seconds — secondary example

**On screen:** Select Combinatorics and compute C(5,3).

**Say:** “The same evidence contract works for combinatorics. Compute first. Explain with citations. Verify every claim.”
