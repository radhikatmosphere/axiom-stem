# AXIOM — Voiceover script (English, ~90 seconds)

Target: 150–170 words at 110–120 wpm = ~80–90 seconds of speech.
Read at a calm, confident pace. Pause 0.5–1 s at each slash `/`.

---

## 0–10 seconds — the promise

> "AXIOM computes STEM answers with deterministic code, then uses OpenAI only to explain the verified result."

## 10–30 seconds — compute (primary demo)

> "This is the primary demo: a single-gene genetics cross, Aa times aa.
> Before any AI text appears, AXIOM enumerates the exact two-by-two Punnett grid.
> The genotype probabilities — fifty percent Aa, fifty percent aa — come from direct enumeration, not from a language model."

## 30–45 seconds — inspect evidence

> "The result is packaged as stable facts and steps. Each one has an ID, like genetics-dot-probability-dot-Aa.
> Those are the only evidence IDs the explanation is allowed to cite."

## 45–65 seconds — explain with citations

> "On the server, AXIOM recomputes the deterministic result, sends that record to OpenAI, validates the JSON, and verifies every claim.
> OpenAI adapts the wording to the learner's level and language — it does not recalculate the cross.
> Citation chips link each sentence back to the evidence ledger."

## 65–75 seconds — verify honestly

> "A green Verified badge means every citation is valid and every number is in the deterministic allowlist.
> If a citation is missing, a number changes, or the model output is malformed, AXIOM honestly shows Warnings — or Could not verify."

## 75–85 seconds — web chat

> "AXIOM also ships a chat interface. Ask Aa times aa or C-of-ten-three in plain text — the same deterministic engine answers, with the same verification badge."

## 85–90 seconds — fallback + close

> "No API key? The flow still works — AXIOM shows a clearly labeled deterministic fallback.
> Compute first. Explain with citations. Verify every claim. That's AXIOM."

---

## Total word count
~155 words → ~80 seconds at 115 wpm. Leaves a 10-second buffer for natural pauses.
